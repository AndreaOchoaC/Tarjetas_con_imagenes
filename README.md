# Tarjetas con imágenes
---
Proyecto para el curso de Python Start 2 en donde crearemos tarjetas con imágenes y texto. 
La tarjeta será una ventana de aplicación creada con PyQt5, con interfaz personalizada (colores, fuente, estilo de botones, etc;). Un botón permitirá que el usuario solicite una imagen y mensaje que añadir a su tarjeta.

Se almacenan imágenes en una carpeta del proyecto (subirla desde el ordenador), posteriormente se muestran de forma aleatoria al hacer click en el botón. Así mismo el usuario crea una lista de mensajes para incluir debajo de la imagen, también de forma aleatoria.

**Mejoras planeadas:**
* Guardar la tarjeta generada como una imagen.
* Agregar texto del usuario a la lista de mensajes.
